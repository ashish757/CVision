# Auth Server

## Overview

The Auth Server is responsible for user authentication, authorization, and session management across the system.

It ensures secure access to resume upload, scoring results, and ranking data.

---

## Responsibilities

- User Registration & Login
- JWT Token Generation & Validation
- Session Management
- Role-Based Access Control (Optional Extension)

---

## Tech Stack

- Node.js
- ExpressJS / NestJS
- PostgreSQL
- Prisma ORM
- Redis (Session Cache)

---

## Future Enhancements
